## 範例 1：精細動作與感覺統合問題

### 輸入
```
家屬表示孩子在學校坐不住，寫字很醜，常常跌倒。
觀察發現：
1. 精細動作：抓握姿勢不成熟，使用全手掌握筆，運筆力道不穩定
2. 粗大動作：無法單腳站立超過3秒，平衡感較差
3. 感覺統合：對觸覺刺激敏感，不喜歡碰觸黏土或沙子
```

### 輸出
```markdown
### 問題分析
1. 精細動作：工具使用經驗不足，運筆技巧處初階階段，手部力量控制尚未成熟
2. 粗大動作：平衡能力發展遲緩，單腳站立穩定度不佳，影響日常活動安全性
3. 感覺統合：觸覺防禦反應明顯，對特定質地材料產生排斥，限制探索學習機會

### 總結與建議
1. 綜合以上結果，建議安排職能療育課程。
2. 精細動作訓練：
   - 透過操作不同粗細的工具（如粗蠟筆、三角鉛筆、細彩色筆）循序漸進練習握筆，從大肌肉控制逐步過渡到精細動作。家長可觀察孩子是否能從全手掌握逐漸調整為三指握，這代表手部小肌肉的分化能力正在進步。
   - 安排穿線板、夾豆子等活動，強化手眼協調與雙手協調能力。這些活動能減少孩子在書寫時因手部控制不佳而產生的挫折感，同時提升日常生活中扣釦子、拉拉鍊等自理能力。
3. 感覺統合調整：
   - 採用漸進式減敏感訓練，從孩子較能接受的材質（如光滑的積木）開始，逐步引入不同觸感的素材（毛巾、米粒、黏土）。過程中尊重孩子的節奏，避免強迫接觸造成反效果。
   - 透過前庭覺刺激活動（如盪鞦韆、平衡木行走），改善整體感覺調節能力，幫助孩子在教室中更能穩定坐姿、專注學習，減少因感覺尋求而離座的頻率。
```

## 範例 2：專注力與社交互動問題

### 輸入
```
老師反應孩子上課無法專心，常常打斷別人說話，不會等待輪流。
觀察發現：
1. 專注力：活動持續時間不超過5分鐘，容易被環境聲音干擾
2. 社交互動：不理解輪流規則，經常插話，眼神接觸少
3. 情緒調節：遇到挫折時容易哭鬧，需要較長時間平復
```

### 輸出
```markdown
### 問題分析
1. 專注力：持續性注意力不足，環境干擾篩選能力較弱，影響學習效率
2. 社交互動：社交規則理解有限，輪流等待概念尚未建立，同儕互動品質受影響
3. 情緒調節：挫折容忍度偏低，情緒恢復時間較長，自我調節策略缺乏

### 總結與建議
1. 綜合以上結果，建議安排短期小團體職能療育課程。
2. 專注力訓練：
   - 使用視覺計時器搭配漸進式延長活動時間的策略，從孩子目前能專注的5分鐘開始，每週增加1-2分鐘。家長可觀察孩子是否能在計時器響起前完成任務，這能幫助孩子建立時間概念與成就感。
   - 在活動中加入「找不同」、「記憶配對」等遊戲，訓練選擇性注意力與視覺搜尋能力。這些能力的提升能減少孩子在課堂中因找不到老師指定的課本頁數而產生的焦慮與挫折。
3. 社交技巧建立：
   - 透過結構化小團體活動（如桌遊、輪流丟球）明確練習「等待」與「輪流」規則。治療師會使用視覺提示卡幫助孩子理解順序，讓抽象的社交規則變得具體可見，降低孩子因不理解而產生的衝突。
   - 運用角色扮演練習適當的對話時機，例如「等別人說完再舉手」。這能幫助孩子在真實情境中減少因插話而被同學排斥的經驗，提升社交參與的信心。
4. 情緒調節策略：
   - 教導深呼吸、數數等簡單的自我安撫技巧，並在孩子情緒激動時提供感覺調節工具（如抱枕、彈力球）。家長可在家中設置「冷靜角」，讓孩子有安全的空間練習情緒管理，減少情緒失控對家庭氣氛的影響。
```
